What I've been working on ...

Portfolio Site

For my first She Codes project, I used HTML and CSS to build this personal portfolio website. The pages are responsive to different devices and the site also contains a working contact form.

Python Project

In the Python module, I wrote functions to process csv files containing data about weather and then convert them into meaningful text-based summaries. All functions had to pass multiple tests involving different scenarios such as blank cells and duplicate values.

Blog Site

For my first Django project I created a blog site. Features include the ability to:

  • create an account
  • create, edit and delete blog posts
  • filter by category on the homepage
  • favourite someone's blog posts

    Crowdfunding Site

    Check Mate is a crowdfunding site built using Django REST framework for the back end and React for the front end. In this site, board game enthusiasts can develop and propose new board games and then ask others for funding support to get the game off the ground. Both the back end and front end were successfully deployed.

    Showcasing Diversity

    A group project where fellow She Coders met with a client who wanted to develop an app to showcase diversity in tech. Using Django and React, we built a site for people who have transitioned into a tech role to share their journeys and connect with others looking to make a similar transition.
